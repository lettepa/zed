Lettepa for [Zed]

## Development

Use [catppuccin/whiskers] to generate the theme file from the `lettepa.tera`
template:

```bash
whiskers lettepa.tera
```

[catppuccin/whiskers]: https://github.com/catppuccin/whiskers
